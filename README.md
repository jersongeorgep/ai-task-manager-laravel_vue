AI Task Manager - Laravel & Vue

An advanced Task Management application built with Laravel and Vue.js, following the Repository Pattern Architecture for clean, scalable, and maintainable development.

Features
Laravel Repository Pattern implementation
Service Layer Architecture
Clean separation of business logic
RESTful API structure
Vue.js frontend integration
Task CRUD operations
AI Service Integration
Architecture

This project demonstrates a professional Laravel development structure where requests travel through multiple layers:

Controller → Service → Repository Interface → Repository → Database

Implemented Components
Task Repository Interface
Task Repository
Task Service
AI Service Layer
AI Integration

Integrated AI providers:

OpenAI
Google Gemini AI

The AI service layer is designed to be extendable, allowing easy integration of multiple AI providers in the future.

Technologies Used
Laravel
Vue.js
MySQL
OpenAI API
Gemini AI API
Purpose

This project is created to demonstrate scalable Laravel application architecture using Repository Pattern and Service Pattern with modern AI integrations.

Backend - Laravel 
Frontend - Vuejs
